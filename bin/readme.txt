Silahkan gunakan program ini untuk merepresentasikan permainan Marble Rolling Toy
1. Buka main.exe
2. Akan muncul tampilan awal sebagai judul program, tekan tombol apa saja untuk melanjutkan.
3. Selanjutnya akan muncul ilustrasi kondisi awal Marble Rolling Toy
4. Masukkan string untuk mencoba permainan, program ini hanya menerima masukan string dengan alphabet A & B.
   Diasumsikan string yang diinput selalu benar, oleh karena itu hati-hati dalam menginput.
5. Setelah input dan enter akan terlihat proses state yang terjadi pada kelereng
6. Kemudian akan tampil state terakhir dan statusnya apakah Diterima atau Ditolak.
7. Setelah selesai program akan menampilkan ilustrasi kondisi akhir Marble Rolling Toy
8. Program selesai
